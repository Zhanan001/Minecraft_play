基于你的第二幕剧本，我将为 **焦灼荒地、霜语山脉、幽影沼泽、晶化洞窟** 四个区域，生成可直接用于 **MythicMobs** 等插件配置的详细怪物规范方案。

### **怪物设计总则**

1. **叙事驱动**：每个怪物的外观、技能、死亡效果都需体现区域的核心悲剧主题。
2. **职业协作**：怪物设计应鼓励不同职业发挥特长（如匠师破甲、元术士AOE、狩魂者控场）。
3. **清晰预警**：所有高伤害或控制技能，必须有明显的粒子效果、音效或前摇动作作为预警。
4. **性能友好**：控制同时存在的怪物数量，避免使用过多持续性的高密度粒子效果。

## **🔥 区域一：焦灼荒地 - 怪物规范**

### **1. 余烬徘徊者**

* **MythicMobs配置ID**：`Ember_Lurker`
* **基础属性**：
  * 生命值：25
  * 伤害：3 (火焰近战)
  * 护甲：2
  * 移动速度：0.25 (缓慢)
  * 类型：`UNDEAD` (亡灵，受亡灵杀手影响)
* **核心技能**：
  1. **余烬灼烧**：近战攻击命中时，有100%概率对目标施加 `燃烧 II`，持续3秒。
  2. **死亡余烬**：死亡时，在自身位置生成一个持续5秒的 `区域效果云`，对进入的玩家每0.5秒造成 `1` 点火焰伤害。
* **AI与行为**：
  * 行为模式：`Neutral` (中立)。在玩家进入其 `FollowRange: 8` 格内被激活。
  * 移动时，身后拖曳细微的灰色灰烬粒子 (`particle: campfire_cosy_smoke`)。
* **刷新**：在副本的外层与中层区域 (`world_scorched_template`) 自然生成。最大同时存在数量：15。

### **2. 熔火畸变体**

* **MythicMobs配置ID**：`Magma_Aberration`
* **基础属性**：
  * 生命值：50
  * 伤害：5 (火焰近战)
  * 护甲：5
  * 移动速度：0.35
  * 类型：`BLAZE` (烈焰人)
* **核心技能**：
  1. **过热冲刺**：每15-20秒，锁定一名玩家并向其直线冲刺，移动速度提升至0.6。冲刺路径上每0.5秒生成一道火焰 (`effect: ignite`)。
  2. **不稳定核心**：死亡时触发。执行指令：`/execute at @self run playsound entity.tnt.primed master @a[distance=..10] ~ ~ ~ 1 1` 播放爆炸音效，并在1.5秒延迟后，对半径3格内所有玩家造成 `10` 点爆炸伤害并击退。
* **AI与行为**：
  * 行为模式：`Aggressive` (主动攻击)。`FollowRange: 12`。
  * 外观：使用资源包，将烈焰人模型替换为熔岩与紫色能量扭曲的形态。
* **特殊刷新**：仅在“净化信标”事件激活时，在信标周围10格内刷新。每波刷新2-3只，共3波。

### **3. 熔核巨像**

* **MythicMobs配置ID**：`Magma_Core_Colossus` (BOSS)
* **基础属性**：
  * 生命值：400 (显示Boss血条)
  * 伤害：10 (常规近战)
  * 护甲：15
  * 移动速度：0.2 (缓慢、沉重)
  * 类型：`GIANT` (巨人，通过资源包替换为熔岩巨像模型)
  * 免疫击退。
* **阶段化技能脚本**：
  * **通用技能**：
    * **熔岩重踏**：每10秒，对前方120度、8格扇形区域发出预警（地面出现1秒的红色粒子带 `particle: dust 1 0 0 1`），随后造成 `15` 点伤害并将该区域转换为 `岩浆块`，持续15秒。
  * **阶段一 (100%-70%)**：
    * 技能循环：熔岩重踏 -> 普通攻击 -> 熔岩重踏。
  * **阶段转换 (70%)**：
    * 喊话：`“过热！焚尽一切！”`
    * 获得新技能：**过热吸引**。
      * 每20秒，以自身为中心，产生持续4秒的强力吸引场。对10格内所有玩家施加 `levitation -0.5` 效果（缓慢拉向中心），并在中心区域持续生成伤害粒子。
  * **阶段二 (70%-30%)**：
    * 技能循环：熔岩重踏 -> 过热吸引 -> 普通攻击。
  * **阶段转换 (30%)**：
    * 喊话：`“核心…不稳定…一起毁灭！”`
    * 获得新技能：**追踪火球**。
      * 每15秒，向随机3名玩家各发射一枚追踪火球 (`skill: fireball`)。火球移动较慢，可被攻击或走位躲避，命中造成 `12` 点爆炸伤害。
    * **狂暴**：攻击速度提升30%。
  * **阶段三 (30%-0%)**：
    * 技能循环：熔岩重踏 -> 过热吸引 -> 追踪火球 -> 普通攻击。技能释放频率加快。
* **AI与行为**：
  * 始终保持对仇恨最高的玩家的面向。
  * 在 `30%` 和 `10%` 血量时，会召唤2只 **余烬徘徊者** 进行干扰。
* **死亡掉落**：必掉 **“破碎的试验日志”** (任务物品) 及职业专属材料。

## **❄️ 区域二：霜语山脉 - 怪物规范**

### **1. 冰元素**

* **MythicMobs配置ID**：`Ice_Elemental`
* **基础属性**：
  * 生命值：20
  * 伤害：4 (远程，魔法)
  * 护甲：0
  * 移动速度：0.3
  * 类型：`SNOWMAN` (雪人)
* **核心技能**：
  1. **冰锥射击**：远程攻击，发射雪球 (`skill: snowball`)，命中后造成伤害并施加 `缓慢 I`，持续3秒。
  2. **冰晶分裂**：死亡时，有100%概率在自身位置召唤2个 **“小型冰元素”** (`MythicMobs ID: Ice_Elemental_Small`)。小型冰元素生命值为10，伤害为2，其他属性相同，但不再分裂。
* **AI与行为**：
  * 行为模式：`Aggressive`，但会尝试与目标保持5-8格距离 (`FollowRange: 12`)。
  * 外观：使用资源包将雪人模型替换为旋转的冰晶聚合体。
* **战术提示**：需要队伍集火秒杀，防止分裂过多导致场面失控。

### **2. 符文冰卫**

* **MythicMobs配置ID**：`Rune_Ice_Guard`
* **基础属性**：
  * 生命值：100
  * 伤害：8 (近战)
  * 护甲：12 (高额减伤)
  * 移动速度：0.18 (缓慢)
  * 类型：`IRON_GOLEM` (铁傀儡)
* **核心技能**：
  1. **寒冰护甲**：正面（以怪物为原点的180度扇形区域）受到的所有伤害降低 `90%`。需通过走位绕至背后攻击。
  2. **寒冰突刺**：每12秒，锁定当前目标，进行短暂蓄力（身上蓝色粒子聚集），然后直线冲锋8格。对路径上的玩家造成 `12` 点伤害并施加 `缓慢 III` (冻结) 2秒。
* **AI与行为**：
  * 行为模式：`Aggressive`，`FollowRange: 10`。
  * 会优先攻击离自己最近的玩家或宠物。
  * 外观：模型为寒铁与蓝冰构成的人形，关节处有发光符文（粒子效果）。
* **战术提示**：狩魂者的宠物是理想的“坦克”，可吸引其正面，让队友安全输出背部。

### **3. 霜语巫祝**

* **MythicMobs配置ID**：`Frostwitch_Specter` (BOSS)
* **基础属性**：
  * 生命值：350 (Boss血条)
  * 伤害：12 (近战冰刺)
  * 护甲：8
  * 移动速度：0.25 (悬浮)
  * 类型：`WITHER_SKELETON` (凋零骷髅，资源包替换为冰封幽魂)
  * 免疫冻结效果。
* **阶段化技能脚本**：
  * **常规技能**：
    * **冰晶射击**：每5秒向目标连续发射3枚跟踪冰弹 (`skill: icemissile`)，单发伤害 `5`。
    * **暴风雪领域**：每15秒，在随机一名玩家脚下召唤一个初始半径2格的蓝色冰霜区域 (`skill: circle`)，区域在5秒内缓慢扩张至4格。停留在区域内的玩家每秒受到 `3` 点伤害并获得 `缓慢 II`。
  * **阶段转换 (50%, 25%)**：
    * 喊话：`“守卫，清除入侵者！”`
    * 在场地两侧召唤2个 **符文冰卫**。
  * **终极技能 - 绝对零度 (10%)**：
    * 喊话：`“归于永恒的静止吧！”`
    * 巫祝进入5秒的不可打断读条状态，全身被极寒白光笼罩。
    * 全场屏幕开始结冰（客户端效果），并在读条结束后，对所有 **未站在‘热能符文圈’内** 的玩家造成 `999` 点真实伤害（即秒杀）。
    * **应对机制**：场地边缘4个固定坐标会提前5秒亮起金色火焰粒子，玩家必须提前进入这些安全圈。
* **AI与行为**：大部分时间悬浮在空中，偶尔落地进行近战攻击。召唤冰卫时会暂时落地。

## **🌫️ 区域三：幽影沼泽 - 怪物规范**

### **1. 腐化藤蔓**

* **MythicMobs配置ID**：`Corrupted_Vine`
* **基础属性**：
  * 生命值：15 (可被任意攻击秒杀)
  * 伤害：3 + 中毒I (接触伤害)
  * 护甲：0
  * 移动速度：0 (静止)
  * 类型：`COD` (鳕鱼，用于创建无实际移动模型的简单实体)
* **核心技能**：
  1. **毒性穿刺**：玩家进入其周围2格范围时，自动触发一次攻击，造成伤害并施加 `中毒 I`，持续5秒。
* **AI与行为**：
  * 行为模式：`Neutral`，无主动AI。
  * 外观：一个微小的、不断扭动的深绿色藤蔓状实体。死亡时播放植物断裂的音效。
* **刷新**：作为环境障碍物，大量、静态地生成在路径、树木和建筑上。

### **2. 低语阴影**

* **MythicMobs配置ID**：`Whispering_Shade`
* **基础属性**：
  * 生命值：10
  * 伤害：1 (接触)
  * 护甲：0
  * 移动速度：0.5 (极快)
  * 类型：`BAT` (蝙蝠)
* **核心技能**：
  1. **恐惧低语**：每5秒，对其当前目标施加 `混乱` 效果（使用 `/effect` 给予 `levitation 1` 并立即取消，或使用插件使玩家随机方向移动1秒）。
  2. **阴影穿梭**：受到伤害后，会瞬间传送到附近5格内的另一个随机位置 (`skill: blink`)。
* **AI与行为**：
  * 行为模式：`Aggressive`，但会不断进行不规则的快速移动，难以被锁定。
  * 外观：半透明的黑色烟雾状，带有两只白色光点作为眼睛。
* **战术提示**：优先用范围技能清理，防止其持续骚扰治疗或施法者。

### **3. 腐化之心守护者 (精英)**

* **MythicMobs配置ID**：`Corrupted_Heart_Guardian`
* **基础属性**：
  * 生命值：150
  * 伤害：12 (近战)
  * 护甲：8
  * 移动速度：0.22
  * 类型：`SLIME` (黏液，尺寸：3，资源包替换为跳动肉瘤)
* **核心技能**：
  1. **毒液喷射**：对前方锥形区域喷射毒液，造成初始 `8` 点伤害，并在区域留下持续10秒的毒液地面，踏入者获得 `中毒 II`。
  2. **心灵尖啸**：每20秒发出尖啸，打断周围10格内所有玩家的施法或引导动作（如元术士引导），并施加 `沉默` 效果3秒（禁用技能）。
  3. **召唤藤蔓**：生命值降至66%和33%时，在自身周围召唤3株 **腐化藤蔓**。
* **AI与行为**：`Aggressive`，移动缓慢但攻击欲望强。是“净化火炬”任务线的关键守卫。
* **死亡掉落**：100%掉落任务物品 **“纯净之心”**。

### **4. 幽影编织者**

* **MythicMobs配置ID**：`Shadow_Weaver` (BOSS)
* **基础属性**：
  * 生命值：300 (Boss血条)
  * 伤害：10 (近战)
  * 护甲：10
  * 移动速度：0.3
  * 类型：`SPIDER` (蜘蛛)
  * 免疫中毒。
* **阶段化技能脚本**：
  * **常规技能**：
    * **毒雾新星**：每15秒以自身为中心释放一次毒雾波，对8格内玩家造成 `5` 点伤害并叠加一层“腐化”减益（可叠加，每层使受到的治疗效果降低10%）。
  * **阶段一 (100%-60%) - 分身戏法**：
    * **阴影分身**：每30秒，制造3个分身。分身具有本体30%的生命值，伤害极低。**真身脚下有持续、明显的紫色粒子漩涡**，分身则没有。
  * **阶段转换 (60%)**：
    * 喊话：`“直面你内心的恐惧吧！”`
    * 获得新技能：**恐惧具现**。
      * 每40秒，随机点名一名非坦克玩家，在其位置召唤一个该玩家的“恐惧幻象”（一个与玩家同外观的黑暗实体）。幻象会攻击队友，需优先转火击杀。
  * **阶段二 (60%-15%)**：
    * 技能循环：毒雾新星 -> 阴影分身 -> 恐惧具现 -> 普通攻击。
  * **阶段三 (15%-0%) - 吞噬黑暗**：
    * 喊话：`“黑暗…融为一体…”`
    * 不再使用分身和恐惧具现。
    * 攻击速度提升50%，移动速度提升30%。
    * 全场环境光逐渐变暗（通过客户端资源包或缓慢的 `blindness` 效果模拟），增加压迫感。
* **AI与行为**：在分身阶段，真身会试图混在分身中。狩魂者的“野兽直觉”技能应能高亮标记真身。

  * 生命值：30
  * 伤害：5 (近战)
  * 护甲：3
  * 移动速度：0.28
  * 类型：`CREEPER` (苦力怕)
* **核心技能**：
  1. **晶体殉爆**：死亡时，身体碎裂，对半径3格内所有玩家造成 `8` 点AOE物理伤害（无视部分护甲）。
* **AI与行为**：
  * 行为模式：`Aggressive`。
  * 外观：使用资源包将苦力怕模型替换为由紫色水晶构成的形态。

### **2. 晶化守卫**

* **MythicMobs配置ID**：`Crystallized_Guardian`
* **基础属性**：
  * 生命值：80
  * 伤害：8 (近战)
  * 护甲：15 (**背部弱点**：来自背后的伤害享受200%加成)
  * 移动速度：0.2 (启动缓慢)
  * 类型：`POLAR_BEAR` (北极熊，模型替换为水晶雕像)
* **核心技能**：
  1. **晶体猛击**：每10秒进行一次重击，对前方小范围造成 `12` 点伤害并施加 `缓慢 II`，持续3秒。
  2. **休眠激活**：初始状态为“休眠”（静止不动，显示为 `slowness 255` 效果）。当玩家进入其警戒范围 (`8`格) 或棱镜被校准后，从休眠中激活（播放水晶碎裂音效）。
* **AI与行为**：
  * 激活后为 `Aggressive` 模式，但移动路径较为固定。
  * 会尝试始终将正面朝向仇恨最高的目标。

### **3. 特殊：记忆水晶 (环境交互实体)**

* **实现方式**：无需MythicMobs。使用 **Citizens2 NPC** 或 **Armor Stand** 配合 **Denizen** 脚本实现。
* **配置要点**：
  1. 在特定坐标（如 `(10,65,20)`）放置一个隐形的盔甲架，命名为 `[Memory_Crystal_A]`。
  2. 当玩家进入其3格范围内，触发Denizen脚本：
     ```yaml
     Memory_Crystal_Trigger_A:
       type: world
       debug: false
       events:
         on player enters area:world_crystal_template,10,65,20,3:
           - if <player.has_flag[heard_crystal_a]> {
               - stop
             }
           - flag player heard_crystal_a
           - playsound <[player]> custom:crystal.whisper.a
           - title <[player]> subtitle “兴奋的男研究员：快看！样本在吞噬‘负面情绪投射体’！过滤效率高达99%！”
     ```

### **4. 特殊：静默蠕虫幼体 (剧情实体)**

* **实现方式**：使用 **Citizens2 NPC**。
* **配置要点**：
  1. `/npc create Silenced_Wormling`
  2. 设置为 `-t SILENT` (安静)，`-p` (可被攻击)。
  3. 绑定Denizen抉择脚本，根据玩家对其的交互（攻击/等待）触发不同的后续任务线，并更新玩家的个人剧情标签 (`/scoreboard players set @s worm_choice 1或2`)。
