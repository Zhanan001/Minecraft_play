## **🔥 区域一：焦灼荒地 - 建筑方案**

### **一、核心理念与总览**

* **主题**：呈现一场由“傲慢科技”引发的、灼热而悔恨的毁灭。建筑是**巨构工程的残骸**。
* **材质主色调**：黑曜石、哭泣的黑曜石、磁石、磨制安山岩（管道）、下界岩、灵魂沙、岩浆块。
* **空间逻辑**：由外向内，从“燃烧的毁灭现场”到“泄露的能量核心”，最终抵达“灾难的源头”。

### **二、坐标蓝图与分区**

假设模板世界 `world_scorched_template` 为边长200格的方形虚空。

```yaml
# 区域边界 (X, Z轴)
外层缓冲区: (-80, -80) 到 (80, 80)
中层污染区: (-50, -50) 到 (50, 50)
核心遗址区: (-25, -25) 到 (25, 25)
# 关键地标坐标 (X, Y, Z)
世界中心/入口点: (0, 65, 0)
净化信标A: (-20, 64, 0)
净化信标B: (0, 64, 20)
净化信标C: (20, 64, 0)
碎星矛残骸基座: (0, 60, 10)
焦灼石碑: (0, 64, 15) # 位于矛基旁
Boss生成点: (0, 66, 10) # 矛基前
```

### **三、关键建筑结构分解**

#### **1. 碎星矛残骸**

* **位置**：`(0, 60, 10)` 至 `(0, 95, 10)`
* **结构**：
  * **基座（Y=60-64）**：5x5的 **哭泣的黑曜石** 平台，中心一格替换为 **命令方块**（伪装），用于触发剧情。
  * **矛身（Y=65-90）**：
    1. 核心柱：`(0, 65, 10)` 至 `(0, 90, 10)` 的 **磁石** 柱。
    2. 外部结构：在核心柱的四个对角线方向（西北、东北、西南、东南），用 **铁块** 和 **金块** 交错搭建，模拟“合金装甲”。结构应呈现扭曲、断裂状，在Y=75、Y=85高度断裂严重。
  * **残破顶部（Y=91-95）**：散乱放置一些 **铁块、金块、哭泣的黑曜石**，模拟炸裂的矛头。
* **特效**：在基座处设置循环命令方块，每2秒执行一次 `/particle minecraft:dust 1 0.2 0 1 ~ ~1 ~ 0.2 0.5 0.2 0 30`（橙红色粒子向上涌）。

#### **2. 能量管道（中层污染区标志）**

* **设计**：巨大、破损的弧形或直线管道。
* **示例管道（从 (-30,65,0) 到 (30,65,0)）**：
  1. **管道底座**：一条3格宽、1格高的 **磨制安山岩** 平台。
  2. **管道壁**：在底座两侧各立一排 **黑色染色玻璃**（高3格），模拟管壁。
  3. **破损**：在中间段移除部分玻璃和底座，形成缺口。
  4. **能量流**：在管道内部（底座上Y=66高度），填充 **紫色染色玻璃** 或 **发光浆果梗**（通过资源包改为紫色），模拟泄露的能量。缺口处用 `紫色粒子（minecraft:dust 0.5 0.0 0.8 1）` 效果溢出。

### **四、环境机制实现**

* **全局灼烧**：在除安全点外的区域，运行循环命令：
  ```
  execute as @a[gamemode=!creative,x=-80,y=60,z=-80,dx=160,dy=30,dz=160,nbt=!{Inventory:[{tag:{scorched_boots:1}}]}] at @s run effect give @s minecraft:fire_resistance 1 0 true
  ```

  *（原理：为未穿戴自定义标签 `scorched_boots` 的玩家每秒刷一次火焰抵抗，抵消伤害，但屏幕可显示火焰动画。也可直接造成伤害。）*
* **紫色污染岩浆**：
  1. 在污染区划定特定坐标区域（如 `(-10,63,-10)` 到 `(10,63,10)`）。
  2. 将该区域的 `lava` 方块替换为自定义方块（如 `light_01`），并通过**资源包**将其材质改为流动的紫色熔岩。

## **❄️ 区域二：霜语山脉 - 建筑方案**

### **一、核心理念与总览**

* **主题**：体现绝对秩序下的冰冷与机械精密感。建筑是**宏伟、对称、无情的巨型设施**。
* **材质主色调**：蓝冰、浮冰、磨制玄武岩、磨制安山岩（齿轮）、铁块、海晶灯、红色染色玻璃（全息投影）。
* **空间逻辑**：从“无序的自然险境（峡谷）”到“有序的人工奇迹（塔楼外部）”，最后进入“秩序的冰冷核心（控制中枢）”。

### **二、坐标蓝图与分区**

```yaml
# 区域边界
凛风峡谷: (-70, -20) 到 (70, 20) # 一条东西向的狭长峡谷
塔楼外部: (-30, -30) 到 (30, 30) # 峡谷中部的山体
控制中枢: (-15, 70, -15) 到 (15, 90, 15) # 位于塔楼顶部
# 关键地标
峡谷入口（西）: (-70, 65, 0)
峡谷出口/塔楼前（东）: (30, 65, 0)
断裂桥梁起点: (20, 70, 0)
断裂桥梁终点: (30, 70, 0)
能源节点1（北）: (25, 68, 15)
能源节点2（南）: (25, 68, -15)
塔楼入口: (30, 70, 0) # 桥梁终点
霜语石碑: (0, 80, 0) # 控制中枢中心
Boss战区域: (-10, 78, -10) 到 (10, 78, 10)
热能符文圈坐标: (12,78,12), (12,78,-12), (-12,78,12), (-12,78,-12)
```

### **三、关键建筑结构分解**

#### **1. 寒霜符文塔楼外部**

* **基座**：以 `(0, 65, 0)` 为中心，用 **磨制玄武岩** 建造一个边长40格、高10格的实心山体基座。
* **塔身**：在基座中央 `(0, 75, 0)` 起，建造一个边长为20格的 **磨制玄武岩** 方塔，向上至 Y=85。
* **巨型齿轮**：在塔身Y=80高度的四个外立面中心，各安装一个巨大的**齿轮**。
  * **齿轮做法**：以立面中心点为圆心，用 **錾制安山岩**（侧放）和 **安山岩墙** 构建一个半径5格、厚1格的圆形齿轮。部分齿可以缺失，呈现破损感。
* **断裂桥梁**：从塔楼入口 `(30,70,0)` 向西搭建一段5格长、3格宽的 **磨制安山岩** 桥，然后断开5格，再在 `(20,70,0)` 位置开始建造另一段桥体，营造需要修复的关卡。

#### **2. 控制中枢内部**

* **布局**：一个充满科技感的环形大厅。
* **中央主控台**：
  1. 在 `(0, 78, 0)` 放置 **霜语石碑**（自定义方块）。
  2. 石碑前方放置一个 **讲台**，上面放一本打开的书（自定义，显示乱码数据）。
  3. 围绕石碑，用 **红石比较器、红石灯、铁活板门** 搭建一个复杂的控制面板外观。
* **全息投影**：在房间一侧的墙上，用 **红色、蓝色、黄色染色玻璃** 拼接成一个不规则的城市俯瞰图。前方用 **海晶灯** 和 **线** 悬挂几个 **物品展示框**，内置指南针，模拟悬浮的操作界面。

### **四、环境机制实现**

* **暴风雪**：在峡谷区域 `(x=-70, y=60, z=-20)` 到 `(x=70, y=80, z=20)` 设置一个 **世界守卫（WorldGuard）区域**，并绑定一个循环任务，每隔90秒触发一次，持续30秒：
  1. `/effect give @a[region=blizzard_region] minecraft:slowness 2 30` （屏幕飘雪效果需资源包支持）
  2. 同时在该区域高密度生成白色粒子：`/particle minecraft:cloud ~ ~ ~ 30 5 30 0 100`
* **热能符文圈**：在Boss战的四个安全点坐标，预先埋设 **压力板** 或 **测重压力板**。当Boss施放“绝对零度”时，通过命令检测站在压力板上的玩家，给予他们免疫秒杀的临时效果。

## **🌫️ 区域三：幽影沼泽 - 建筑方案**

### **一、核心理念与总览**

* **主题**：表现被恐惧与疯狂腐蚀的自然与文明。建筑是**腐朽、混乱、令人不安的**。
* **材质主色调**：深色橡木、云杉木、藤蔓、黏土、苔石砖、发光蘑菇、水、旗帜。
* **空间逻辑**：从“毒雾弥漫的丛林”到“悬于树顶的废弃文明痕迹”，最后沉入“被亵渎的古老神殿”。

### **二、坐标蓝图与分区**

```yaml
# 区域边界
迷雾丛林: (-100, -100) 到 (100, 100)
树顶村落（空中）: Y=80 至 Y=100
下沉神殿: (0, 50, 0) 为中心，半径20格
# 关键地标
副本入口/莱拉营地: (0, 65, -80)
净化火炬A（东）: (40, 85, 0) # 树顶村落
净化火炬B（南）: (0, 85, 40)
净化火炬C（西）: (-40, 85, 0)
神殿入口（上方）: (0, 65, 0) # 此处有向下的通道
幽影石碑: (0, 52, 0) # 神殿最深处
腐化之心守护者刷新点: (0, 65, 20)
Boss战祭坛区域: (-8, 51, -8) 到 (8, 51, 8)
```

### **三、关键建筑结构分解**

#### **1. 树顶村落**

* **巨树**：在关键坐标点（如火炬点），用 **深色橡木原木** 建造数棵直径5-7格、高20-30格的巨型树木作为支撑柱。
* **树屋平台**：在Y=80-90的高度，用 **云杉木板** 和 **云杉木台阶** 在巨树间搭建不规则的、相互连通的平台。部分平台应倾斜、破损，用 **云杉木栅栏** 和 **绊线** 模拟破损的栏杆。
* **吊桥**：用 **云杉木栅栏** 作为索链，**云杉木板** 作为桥面，连接相距较远的平台。中间段可断开。
* **装饰与叙事**：
  * **图腾**：在平台上立起 **云杉木栅栏**，顶部放上 **各种生物的头颅**，周围用 **旗帜** 和 **蜡烛** 装饰。
  * **生活痕迹**：散放 **炼药锅**（盛有绿色水）、**堆肥桶**、**破损的床**。

#### **2. 下沉神殿**

* **入口**：在 `(0, 65, 0)` 创建一个向下的垂直通道，用 **苔石砖楼梯** 螺旋向下，通道内布满 **藤蔓** 和 **蜘蛛网**。
* **神殿主厅**：
  * **墙体**：用 **苔石砖、裂石砖** 建造一个20x20的方形大厅，高度10格。
  * **立柱**：厅内有4根 **石砖** 立柱。
  * **积水**：将地面Y=51层全部替换为 **水源**，营造半淹没感。在水下放置 **海晶灯**（调低亮度）提供幽暗照明。
  * **祭坛与石碑**：在大厅尽头 `(0, 52, 0)` 建造一个3级高的 **苔石砖台阶** 作为祭坛，石碑立于其上。祭坛周围用 **黑色染色玻璃** 围住，Boss死亡后玻璃消失。

### **四、环境机制实现**

* **毒雾**：在整个丛林和神殿区域，使用 **`/effect`** 命令结合 **区域效果云（Area Effect Cloud）** 实体，持续给予玩家 `中毒I` 效果。紫色毒雾区域则提升为 `中毒II` 并附加 `反胃`。
* **腐化藤蔓**：这些“怪物”实际上可以是一种**自定义的刷怪蛋**，由管理员在搭建时手动放置在关键路径上，看起来就像环境的一部分。
* **精神干扰（幻影）**：在神殿区域，使用循环命令方块随机执行：
  ```
  execute at @a[tag=in_temple] run summon minecraft:illusioner ~ ~5 ~ {PersistenceRequired:1,NoAI:1,Invulnerable:1,Silent:1,ActiveEffects:[{Id:14,Duration:1,ShowParticles:0b}]}
  execute as @e[type=illusioner] at @s run tp @s ~ ~5 ~
  execute as @e[type=illusioner] at @s run kill @s
  ```

  *（这条命令会瞬间在玩家头顶召唤一个隐身且无敌的幻术师，并立即将其传送到更高处杀死，形成一闪而过的黑影效果。）*

## **💎 区域四：晶化洞窟 - 建筑方案**

### **一、核心理念与总览**

* **主题**：展现神圣、静谧却又充满悲怆记忆的科技圣所。建筑是**晶体、精密仪器与静谧空间的结合**。
* **材质主色调**：紫水晶母岩、紫水晶块、紫水晶簇、方解石、磨制安山岩、玻璃、海晶灯、水。
* **空间逻辑**：从“自然形成的晶体回廊”进入“储存记忆的水晶森林”，穿过“人工的实验室前厅”，最终抵达进行终极仪式的“核心实验室”。

### **二、坐标蓝图与分区**

```yaml
# 区域边界（地下结构，入口在(0,65,0)）
共鸣回廊（上层）: Y=40 至 Y=60
记忆水晶林（中层）: Y=20 至 Y=40
实验室前厅（下层）: Y=10 至 Y=20
核心实验室（底层）: Y=0 至 Y=10
# 关键地标

冰之棱镜（北）: (0, 12, 25)
雷之棱镜（西）: (-25, 12, 0)
记忆回响装置: (0, 5, 0)

静默蠕虫出现点: (0, 6, 5)
```

### **三、关键建筑结构分解**

#### **1. 记忆水晶林**

* **环境**：一个巨大的天然洞窟。穹顶和地面布满 **紫水晶母岩**。
* **巨型晶簇**：
  * 使用 **紫水晶块** 作为晶簇主干，从地面或穹顶生长。
  * 用 **紫水晶簇** 密集地装饰在主干表面。
  * 使用 **淡灰色、灰色染色玻璃** 包裹部分晶簇，形成“包裹记忆”的视觉效果。
  * **中央湖泊**：在 `(0,25,0)` 挖出一个直径15格的圆形湖泊，底部铺 **黑色混凝土**，中心放置一个 **光源方块（海晶灯）** 向上打光，营造神秘与静谧感。

#### **2. 记忆回响装置**

* **位置**：核心实验室中央，`(0, 5, 0)`。
* **结构**：
  1. **基座**：一个3x3的 **紫水晶块** 平台。
  2. **能量导管**：从基座四角向上延伸3格高的 **紫水晶块** 柱。
  3. **悬浮核心**：在基座中心上方Y=7格处，悬空放置一个 **紫水晶簇**（使用 `armor_stand` 或 `falling_block` 定格实现）。核心周围环绕4个 **海晶灯**。
  4. **环绕棱镜**：在能量导管顶部（Y=8），四个方向各水平延伸出一根由 **玻璃** 和 **海晶灯** 构成的“光路”，连接至墙上的四个接收器。
* **修复交互点**：在基座旁放置一个自定义命名的 **`dropper`（发射器）**，外观通过资源包改为破损的仪器。玩家提交修复材料实则是向发射器内放入特定物品。

### **四、环境与交互机制实现**

* **共鸣棱镜校准**：
  1. 三个棱镜点各放置一个名为 `[Fire_Lens]` 等的 **盔甲架**，手持自定义的棱镜物品。
  2. 当玩家持有“符文扳手”右键时，触发一个 **快速时间事件（QTE）**。这可以通过 **点击GUI插件**，或在一系列浮空文字上要求玩家在限定时间内点击对应顺序的按钮来实现。
* **记忆水晶低语**：在特定的记忆水晶坐标旁，放置不可见的 **`marker`** 实体。当玩家接近时，通过命令播放对应音频（`/playsound`）并在其屏幕显示字幕（`/title actionbar`）。
* **静默力场**：在核心实验室区域（Y=0-10）使用 **WorldGuard** 设置区域标志 `-blocked-cmds list`，禁止玩家使用职业技能的相关指令（如 `/skill` 等）。

### **核心执行摘要**

1. **模板世界**：所有建筑必须在四个独立的 **`world_*_template`** 模板世界中预先完美搭建完成。
2. **坐标即契约**：所有任务触发器、NPC生成点、怪物刷怪点必须与上述蓝图坐标严格对应。
3. **机制预埋**：在建造时，需同步埋设命令方块、红石电路、区域标记等，以实现环境机制和剧情触发。
4. **资源包依赖**：第二幕的视觉（紫色岩浆、水晶纹理）和听觉（记忆低语、环境音效）体验高度依赖自定义资源包，需同步开发。
