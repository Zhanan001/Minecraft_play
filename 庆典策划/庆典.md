# **第一赛季收官庆典详细策划方案**

## **“铭记纪元，共赴深蓝”**

---

## **一、 庆典核心目标**

1. **情感沉淀**：提供一个正式、有仪式感的场景，让玩家回顾并告别共同经历的史诗赛季。
2. **社区凝聚**：通过轻松、合作的非对抗性活动，强化玩家间的社交纽带，庆祝共同的胜利。
3. **赛季过渡**：自然平滑地将玩家注意力从“符文纪元”引向“深海之歌”，激发对新赛季的期待。
4. **数据回收与反馈**：通过活动温和地回收赛季货币，并为运营团队收集玩家行为与意见。

---

## **二、 庆典基本信息**

- **名称**：第一赛季“符文觉醒”纪元庆典
- **时间**：第67天 00:00 至 第70天 24:00 （现实时间，共4天）
- **主题**：根据服务器最终结局线路而定：
  - **A线（永恒守望）**：主题为 **“静默的盛宴与未来的微光”**，氛围庄严、宁静、充满回忆。
  - **B线（破晓新生）**：主题为 **“破晓的庆典与远航的序曲”**，氛围欢快、热烈、充满希望。
- **主会场**：
  - A线：**“永恒监视者”方尖碑**广场及周边水晶纪念碑园区。
  - B线：**“世界之树”** 根部广场及周边新生林地。

---

## **三、 详细活动内容与执行脚本**

### **▌ 活动1：全服庆典启动仪式（第67天 20:00）**

* **形式**：全服同步线上事件。
* **流程**：
  1. **19:50**：全服公告预热：`「庆典即将开始，请各位守护者/先驱前往主会场！」` 主会场播放舒缓的背景音乐。
  2. **20:00**：服务器核心NPC（阿尔图/塞拉）发表简短感谢致辞（全服弹窗文本）。
  3. **20:05**：**“纪元光影秀”** 开始。利用 `ProtocolLib` 和 `EffectLib`，在主会场上空进行持续5分钟的、展现本赛季关键剧情节点（如职业觉醒、四大石碑、最终决战）的粒子灯光秀，并配以赛季主题旋律。
  4. **20:10**：光影秀结束，宣布所有庆典活动正式开启。主会场刷新第一批 **“庆典招待员”NPC**，提供活动指引。

### **▌ 活动2：赛季回忆长廊（常驻，全庆典期）**

* **形式**：静态展示与互动体验。
* **内容**：
  1. **全息历史墙**：在主会场一侧，使用 `Holographic Displays` 生成多列全息画面，滚动展示：
     - 由系统自动抓取的、本赛季玩家建造的**优秀建筑截图**。
     - 四大区域**首通队伍的名单与时刻**。
     - 击败最终Boss的 **“最后一击”玩家特写**。
  2. **故事典籍亭**：放置一个书架NPC，玩家可右键浏览一本名为 **《艾瑟拉纪元大事记》** 的书籍，里面以编年史形式记录了从开服到结局的所有主要世界公告和事件描述。
  3. **“我的赛季时刻”投稿箱**：设立一个邮箱NPC，玩家可使用游戏内的“书与笔”写下自己的赛季感想或故事，并署名投稿。优秀投稿将被收录，并在下赛季的某个角落永久展示。

### **▌ 活动3：全服协作——建造“纪元丰碑”（第67-69天）**

* **形式**：全服玩家共同完成的开放式建造项目。
* **设计**：
  - **A线**：在静默深渊旁，建造一座 **“铭记之墙”**，墙上可留名。
  - **B线**：在世界之树旁，建造一座 **“希望之港”** 的微缩模型，寓意远航。
* **玩法**：
  1. 场地提供专用的 **“庆典建材包”**（包含各种彩色混凝土、玻璃、灯饰等非实用装饰方块）。
  2. 玩家可自由领取建材，在规划区域内（用粒子效果标出）进行建造。
  3. 使用 `WorldEdit` 的 `//copy` 和 `//paste` 功能，由在线管理员或可信玩家将小型个人创作整合到主结构上。
  4. 项目完成后，全服玩家获得共同成就 **“共同记忆的铸造者”** 及特殊称号。

### **▌ 活动4：趣味挑战赛（第68-69天，每日轮换）**

* **形式**：在副本世界 (`Multiverse`) 内举行的限时趣味小游戏竞赛。
* **每日安排**：
  - **第68天 14:00-16:00**：**“方块速建王”**。玩家进入一个空白房间，根据给出的主题（如“赛季最难忘的场景”），在5分钟内用提供的方块进行快速建造，由系统或评委投票选出最佳。
  - **第68天 20:00-22:00**：**“符文知识竞赛”**。通过 `QuizPlugin` 插件，在全服频道进行赛季剧情、设定相关的趣味问答，前10名答对最多的玩家获得奖励。
  - **第69天 14:00-16:00**：**“和平的竞速赛”**。在布满障碍和趣味机关的赛道里进行跑酷比赛，禁止任何PVP。
  - **第69天 20:00-22:00**：**“解谜寻宝”**。在主世界设置一系列与赛季剧情相关的谜题线索，引导玩家寻找隐藏的“庆典宝箱钥匙”。

### **▌ 活动5：庆典限定福利与兑换（全庆典期）**

* **形式**：NPC商店与任务奖励。
* **内容**：
  1. **“纪元商人”**：出售限时庆典物品，使用赛季货币或新的“庆典券”购买。
     - **必购品**：限定烟花、头盔饰物（如庆典礼帽）、纪念横幅。
     - **珍稀品**：**赛季相框**（可将玩家自定义图片放入游戏内展示）、**纪念宠物**（如一只发光的小水晶虫或一只幼年海豚）。
  2. **“感谢任务”**：一系列极其简单的日常任务（如与10位玩家交换物品、在主会场跳舞等），完成后给予大量“庆典券”和经验。
  3. **抽奖轮盘**：使用“庆典券”进行抽奖，奖品包含普通建材、稀有幻化、及极低概率的 **“开发者签名物品”**（一件命名有开发者ID的纪念品）。

### **▌ 活动6：静默告别与未来寄语（第70天，A线专属）/ 破晓狂欢与远航仪式（第70天，B线专属）**

* **形式**：结局路线专属的收官仪式。
* **A线执行脚本（晚8点）**：
  1. 所有玩家被邀请到“静默深渊”边。
  2. 阿尔图进行简短演说，感谢大家的坚守。
  3. 玩家使用任务物品 **“静默之花”**，右键投入深渊，作为对囚徒的最终告别与对牺牲的铭记。
  4. 全服静默1分钟，只有风声和柔和的背景音乐。
  5. 公告：`「长夜漫漫，守望永续。愿这份宁静，能孕育出新的勇气。」`
* **B线执行脚本（晚8点）**：
  1. 所有玩家被邀请到“希望之港”。
  2. 塞拉进行激昂演说，庆祝新生，并眺望远方的大海。
  3. 举行“放流祈愿灯”活动，玩家将写有愿望的纸船（游戏内物品）放入新开凿的运河中，流向大海。
  4. 盛大的终场烟花表演，结合“深海之歌”主题的蓝绿色调。
  5. 公告：`「旧章已落，新篇将启。破晓之光，必将照亮深邃的海渊！」`

### **▌ 活动7：下赛季预热——“深海的呼唤”（第70天全天）**

* **形式**：环境叙事与物品暗示。
* **执行**：
  1. **环境变化**：在所有水域（尤其是海边）生成淡淡的、梦幻的**蓝色雾气粒子**。夜晚偶尔能听到从海洋方向传来的、空灵的鲸歌（自定义音效）。
  2. **物品获取**：完成所有庆典日常任务的玩家，必定获得一个 **“神秘漂流瓶”**。打开后获得一张字条，上面写着一段 cryptic 的诗句或航海坐标（即下赛季登录点的预告），以及一件 **“未鉴定的深海遗物”**（第二赛季开服后才可鉴定）。
  3. **最终公告**：在服务器关闭维护前最后一刻发布：`「海浪轻抚着崭新的船帆，深渊之中，古老的歌谣已然唱响。第二赛季『深海之歌』，即将于[日期]破浪而来。敬请期待！」`

## **四、 技术实现与运营清单**


| 活动模块            | 所需核心插件/工具                                 | 配置与执行要点                                                        | 负责人      |
| :------------------ | :------------------------------------------------ | :-------------------------------------------------------------------- | :---------- |
| **光影秀/粒子特效** | `ProtocolLib`, `EffectLib`, `ParticleLib`         | 预先编写好粒子序列代码，测试性能。准时触发命令。                      | 技术组      |
| **NPC对话与商店**   | `Citizens`, `Shopkeepers`, `Quests`               | 编写所有NPC对话文本，配置商店物品和价格，设置简单任务链。             | 内容组      |
| **全息投影与展示**  | `Holographic Displays`                            | 收集并筛选玩家截图，制作全息文本和图片序列。                          | 美术/社区组 |
| **趣味挑战赛**      | `Multiverse`, `MinigamesLib` (可选), `QuizPlugin` | 提前搭建好比赛用副本世界，配置好跑酷地图和谜题。                      | 活动组      |
| **全服建造管理**    | `WorldEdit`, `WorldGuard`                         | 划定建造区，设置玩家权限（仅允许放置/破坏特定方块组），定期备份整合。 | 建筑/管理组 |
| **专属物品与兑换**  | `ItemsAdder` 或 `CustomItems`, `Vault`            | 设计并导入庆典限定物品的纹理与数据，配置经济兑换。                    | 技术/美术组 |
| **音效与环境**      | `SoundPacket`, 资源包                             | 准备鲸歌等新音效文件，制作简单的雾气资源包。                          | 技术组      |
| **公告与统筹**      | `DiscordSRV`, 服务器定时任务                      | 在Discord与游戏内同步发布活动时间表，准时触发各项活动。               | 服主/运营   |
