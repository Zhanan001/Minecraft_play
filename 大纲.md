# **第一赛季：符文觉醒与遗忘纪元 - 导演终极版执行剧本**

## **🎬 核心设计总纲：史诗叙事与游戏机制的原子级融合**

本方案将第一赛季70天（10周）的旅程，分解为**4幕史诗、10个章节、31个关键剧情日**的逐日执行手册。每个任务对话、BOSS技能、世界变化都已转化为可直接配置进 **Quests、MythicMobs、Citizens** 等插件的详细参数。

### **🌍 世界观终极设定：颠覆性的“真相”**
**核心反转**：玩家发现的“静默蠕虫”并非毁灭上古文明的元凶，而是文明为封印真正的“灾难”——**自身无限膨胀的欲望所化的“熵灵”**——而创造的“活体监狱”。蠕虫吞噬符文能量，是为了不让能量被熵灵利用。上古领袖启动“大静默”，是悲壮地将蠕虫与熵灵一同封印。

**情感内核**：本季结局不是简单的击败邪恶，而是在**理解与牺牲**中做出选择：是杀死尽职的“监狱”（蠕虫）释放未知的邪恶（熵灵），还是找到第三条路？

## **📖 第一幕：迷雾初开 (第1-2周) - “我们从灰烬中睁眼”**

### **第1周：流浪者之日**
**环境状态**：全局光照设置为苍白色，偶尔有灰色粒子雨。出生点“宁静河谷”布设烧焦的树木、残缺的石雕。

**关键剧情日执行脚本**：

**> 第1天 - 开服日**
*   **0700（现实时间）**：服务器开启。
*   **玩家出生动作**：立即获得持续5分钟的“迷茫”效果（缓慢I）。背包中有一本**《浸水的日记 - 第7页》**（通过插件在首次登录时自动给予），文字模糊，仅可辨认：“…河谷…最后一个安全区…不要相信闪烁的影子…”
*   **世界公告**（黄色，加粗）：`「从漫长的静默中，你们苏醒了。空气沉重，记忆破碎，但河谷的庇护尚存——暂时如此。」`
*   **任务链自动开始**（Quests插件）：
    1.  **任务名**：《搜集篝火材料》
        *   **目标**：收集橡木原木x10、煤炭x5。
        *   **完成对话**（完成时弹出）：“篝火带来了温暖，也带来了…注视。你感到黑暗中有东西在移动。”
    2.  **任务名**：《异常的声响》
        *   **目标**：击杀夜晚在河谷边缘生成的“**畸变蝙蝠**”（MythicMobs配置：血量10，攻击附带3秒失明，掉落“变异的翼膜”）。
        *   **完成对话**：“蝙蝠的形态很不自然，它们的体内有微弱的…紫色能量脉络。”

**> 第3天 - “方尖碑的脉动”**
*   **环境变化**：20:00整点，服务器时间强制设为夜晚。出生点中央的“远古方尖碑”（由哭泣的黑曜石、磨制安山岩搭建）开始发出频率为2秒/次的低沉心跳声（全域可闻的声音事件），并散发紫色粒子效果。
*   **新增任务链**：
    1.  **任务名**：《心跳调查》
        *   **目标**：右键点击方尖碑。
        *   **完成对话**（弹出书籍GUI）：
            ```
            **碑文触感反馈**：
            你的手触碰碑体的瞬间，一股冰冷的脉冲顺手臂蔓延。
            脑海中炸响一个词：**“收集…碎片…共鸣…”**
            碑体底部滑出一个石匣，里面有三块「黯淡的符文石」。
            ```
    2.  **任务名**：《碎片的共鸣》
        *   **目标**：在方尖碑脉冲时（每2秒），手持符文石右键点击，完成0/3次充能。
        *   **机制**：充能时玩家会被固定1秒，并受到“能量灌注”效果（发光I）。
        *   **完成对话**：“符文石变得温暖，仿佛在与你血脉相连。它们渴望…更多的同类。”

### **第2周：觉醒之周**
**环境状态**：河谷外的“灰寂森林”区域开放，森林中随机生成散发微光的“符文萤石”，作为新资源点。

**关键剧情日执行脚本**：

**> 第8天 - “职业觉醒仪式”**
*   **前置条件**：全服累计充能碎片达100块时触发。
*   **事件序列**：
    1.  **20:00**：全服播放音效“ENTITY_ENDERDRAGON_GROWL”（低沉龙吟）。屏幕中央显示倒计时10秒。
    2.  **20:00:10**：所有在线玩家被强制传送至方尖碑周围20格内。屏幕渐黑。
    3.  **过场动画**（通过Holographic Displays生成全息文字，配合Title公告）：
        *   **第一行**（紫色，大号）：`「能量阈值突破——」`
        *   **第二行**（白色）：`「上古回响正在解码…」`
        *   **第三行**（金色）：`「选择你的道路，流浪者。」`
    4.  **职业选择**：玩家面前弹出**虚拟菜单**（使用DeluxeMenus插件配置）：
        *   **选项1 - 匠师**：[铁砧图标] “重塑物质，铸造永恒。”
        *   **选项2 - 元术士**：[末影之眼图标] “驾驭流动，洞悉本质。”
        *   **选项3 - 狩魂者**：[狼牙图标] “聆听荒野，羁绊生命。”
        *   **选择后**：播放对应的技能特效（匠师：铁砧敲击声与火花；元术士：奥术矩阵环绕；狩魂者：狼嚎），并永久授予职业标签。
*   **觉醒后任务**：
    *   **匠师**：《第一把刻刀》 - 制作“符文刻刀”（木棍+铁锭+充能符文石），用于解锁专属合成。
    *   **元术士**：《能量感知》 - 使用技能“能量视觉”（右键空气）找到3个隐藏的“地脉节点”。
    *   **狩魂者**：《荒野伙伴》 - 驯服一匹“森林狼”（特殊变种，血量更高）。

## **📖 第二幕：裂痕重现 (第3-5周) - “石碑低语着被掩盖的历史”**

### **第3-4周：四域探险
**关键剧情日执行脚本**：

**> 第15天 - “焦灼荒地的警告”**
*   **区域特性**：该区域永久燃烧，玩家需饮用“抗火药水”（元术士可集体制造降温结界）。荒地中央矗立“焦灼石碑”。
*   **石碑解读事件**：
    1.  玩家右键石碑，播放动画：地面裂开，喷出火焰柱，在空中形成一幅**全息残影**——古代城市被巨大的能量光束摧毁。
    2.  **播放语音**（使用音频插件，沉稳男声，带混响）：“…我们犯下了傲慢之罪。符文之力本应构筑，我们却用它制造了‘碎星矛’。当第一座城市因测试而蒸发时，欢呼变成了哀嚎…”
    3.  **获得物品**：“灼热的记忆残像”（任务物品，用于触发后续剧情）。
    4.  **全服公告**：`「焦灼的记忆已灼伤一位探索者的心智。历史并非总是荣耀。」`

**> 第22天 - “晶化石碑的牺牲”**
*   **区域特性**：洞窟内布满水晶，需匠师制造“共振镐”高效采集。深处为“晶化石碑”。
*   **石碑解读事件**：
    1.  动画：水晶发出共鸣，投射出**议会场景**——一群身着长袍的人正在激烈争论。
    2.  **播放语音**（苍老女声）：“…蠕虫是我们创造的！它是吞噬失控能量的‘清道夫’，但民众视它为怪物…熵灵在利用恐惧分化我们。启动‘大静默’是唯一的选择，代价是我们的文明…我们自愿成为罪人。”
    3.  **关键转折**：此时，地面钻出一只小型“静默蠕虫幼体”（血量500），但它**不主动攻击**，而是开始吞噬周围水晶的能量。如果玩家攻击它，它会以最低伤害反击，并发出悲伤的嘶鸣。如果玩家在1分钟内不杀死它，它会吞噬完能量后，掉落一个“纯净的能量结晶”并消失。
    4.  **全服公告**：`「晶格的记忆揭示了创造与监禁。也许怪物，亦有它被赋予的使命。」`

## **📖 第三幕：囚徒与狱卒 (第6-8周) - “我们该站在哪一边？”**

### **第6周：地脉核心的真相**
**关键剧情日执行脚本**：

**> 第36天 - “蠕虫幼体遭遇战”**
*   **副本入口**：方尖碑底部开启传送门。
*   **BOSS：静默蠕虫幼体**（MythicMobs详细配置）
    *   **血量**：15,000
    *   **阶段一（100%-70%）**：
        *   技能1：“能量虹吸” - 随机锁定一名玩家，每秒吸取其3点饥饿值，转化为自身治疗。
        *   **职业应对**：元术士使用“能量干扰”技能（右键对准BOSS）可打断此吸食。
    *   **阶段二（70%-40%）**：
        *   技能2：“静默领域” - 在随机玩家脚下生成一个5x5的紫色领域，领域内玩家**所有职业技能失效**，持续15秒。
        *   **职业应对**：匠师可向领域内投掷“共鸣炸弹”（提前制造），提前驱散领域。
    *   **阶段三（40%-0%）**：
        *   技能3：“悲鸣” - 释放全屏音波攻击，造成伤害并施加“脆弱”效果。
        *   **机制**：场地边缘刷新“上古符文柱”，狩魂者引导其驯服的生物触碰符文柱，可为全队生成一个持续6秒的“灵魂护盾”，完全免疫此次攻击。
*   **击败后**：不掉落装备，而是掉落 **“蠕虫的记忆腺体”**。右键点击播放**最终记忆闪回**：
    *   **画面**：无数蠕虫温顺地围绕在符文工程师身边，吞噬着从城市中抽离的、带有暗红色（熵灵污染）的能量。
    *   **语音**（蠕虫的集体心念，空灵而悲伤）：“…我们…被塑造…为了‘吞噬’…为了‘容纳’…我们关着‘它’…也关着自己…‘静默’是永恒的牢笼…我们…都…想…自由…”

### **第7周：艰难抉择与全服公投**
*   **任务**：《抉择之时》
    *   玩家将“蠕虫的记忆腺体”带回主城，交给NPC“大法师阿尔图”（Citizens插件）。
    *   **阿尔图对话**：“…真相令人窒息。我们一直对抗的，是守护我们的狱卒。但现在，封印在弱化，‘熵灵’可能在蠕虫死后逃脱。两个选择：**强化封印**（永久禁锢蠕虫与熵灵），或**净化释放**（杀死蠕虫，并集全服之力正面迎战熵灵）。”
*   **全服公投系统**（使用投票插件）：
    *   投票持续48小时。
    *   每个城镇（Towny）根据等级获得不同票数，由镇长代表投票。
    *   投票过程及结果，通过DiscordSRV实时直播。

## **📖 第四幕：纪元终章 (第9-10周) - “我们的选择，我们的纪元”**

### **根据第7周公投结果，产生两条完全不同的剧情线：**

#### **【A线：选择“强化封印” - “永恒的守望者”】**
**> 第50天 - “封印仪式”**
*   **任务**：各城镇需贡献资源，在方尖碑处建造“永恒监牢”（大型多方块结构）。
*   **最终事件**：仪式启动时，巨大的蠕虫本体（千米级，作为环境动画）从地底浮现，平静地凝视玩家，然后缓缓沉入新生成的、被水晶封印的深渊中。
*   **结局动画**：世界恢复色彩，但方尖碑永远变为暗紫色。所有参与玩家获得称号 **“寂静守护者”**、专属的“封印之链”背部装饰。
*   **结局公告**：`「你们选择了责任与牺牲。艾瑟拉重获宁静，但每一位守护者都知道，这份宁静之下，沉睡着两个永恒的囚徒。一个新的、务实而沉重的纪元开始了。」`

#### **【B线：选择“净化释放” - “破晓的赌徒”】**
**> 第50天 - “最终决战：熵灵显现”**
*   **BOSS：熵灵·无尽饥渴**（MythicMobs终极配置）
    *   **形态**：不断变换形态的暗红色实体。
    *   **核心机制**：“欲望汲取” - BOSS会复制玩家职业的终极技能来攻击玩家。玩家必须在被复制的技能击中前，**切换成另一种职业的基础攻击模式**，使BOSS的复制系统“过载”进入短暂瘫痪。
*   **最终阶段演出**：击败熵灵后，蠕虫的残存意识化为光点，治愈满目疮痍的大地。熵灵消散后，留下一颗“纯净的纪元之种”。
*   **结局任务**：《种植未来》 - 全服玩家护送“纪元之种”至世界中心，种下。瞬间，一棵贯穿天地、散发柔和光辉的巨树生长而出。
*   **结局动画**：巨树的枝叶净化了整个世界的腐化区域，新的、从未见过的植物与温和生物涌现。
*   **奖励**：所有玩家获得称号 **“破晓先驱”**、可飞行的“光翼”幻化。
*   **结局公告**：`「你们选择了希望与风险。以旧日守护者的牺牲为基石，你们击败了纯粹的贪婪，种下了全新的未来。这是一个高风险的、光辉灿烂的新纪元。」`

### **> 第70天 - 赛季收官庆典**
*   无论何种结局，最后三天为庆典。
*   **活动**：全地图刷新“纪念宝箱”，内含食物、烟花、以及**下赛季“深海之歌”的预告物品**（如“潮湿的船票”、“古老的海图碎片”）。
*   **开发者信**：在所有主城以告示牌形式出现，感谢玩家，并公布第二赛季的精确开启时间。
