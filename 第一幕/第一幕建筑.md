# 《迷雾初开》第一幕建筑实现策划

## 一、总体设计原则

### 1. 建筑风格定位

- **基调**：后末日/废土风格 + 古代遗迹神秘感
- **色彩**：灰、白、黑为主色调，点缀紫色能量光效
- **材质**：破损石材、烧焦木材、风化金属、发光晶体
- **规模**：中小型建筑，避免宏大规模（符合避难所设定）

### 2. 功能与叙事结合

- 每个建筑都应有背景故事痕迹
- 环境叙事：通过破损程度、物品摆放讲述过去
- 功能性建筑同时承担剧情引导作用

## 二、核心建筑详细设计

### 1. 中央集会广场 (30×30格)

#### **整体布局**

```
            [北]
    石雕废墟     石雕废墟
        \         /
         \       /
          [中央篝火]
         /       \
        /         \
    任务板       布雷恩位置
            [南]
```

#### **篝火区** (中心区域 5×5)

- **底座**：圆形，直径5格
  - 底层：裂石砖+苔石砖混合铺地
  - 中层：圆石台阶围成圆形火坑
  - 内部：灰烬块（1格厚）+ 灵魂沙（凹陷效果）
- **篝火结构**：
  - 中央支柱：烧焦的橡木原木x3（垂直堆叠）
  - 顶部：营火（永久燃烧状态）
  - 周围：随意放置的木柴（木棍、木板散落）
- **光照系统**：
  - 主光源：营火（光照等级15）
  - 辅助：4个火把（东南西北四方向，离地2格高）
  - 特效：使用粒子效果生成缓慢上升的灰烬粒子

#### **布雷恩位置** (东南角)

- **破损木箱**：
  - 材质：橡木木板+云杉木板混合（破损感）
  - 结构：2×1×1大小，盖子半开
  - 内饰：内部可见少量物品（腐肉、木棍模型）
- **倚靠的铁剑**：
  - 使用物品展示框固定在箱侧
  - 材质：铁剑+铁砧重命名显示"生锈的铁剑"
  - 角度：45度倾斜
- **布雷恩活动区域**：
  - 站立点：裂石砖+骨块（暗示守卫身份）
  - 周围装饰：空瓶子x2、皮革碎片（地毯）、小石子（安山岩压力板）

#### **任务公告板** (西南角)

- **结构**：
  - 支架：2根橡木原木（烧焦效果）
  - 板面：橡木告示牌（3个纵向排列）
  - 内容：
    - 顶部："河谷安全守则"
    - 中部："今日任务：[破损文字]"
    - 底部："保持篝火燃烧！"
- **地面装饰**：
  - 散落的纸片：白色地毯
  - 掉落羽毛：羽毛物品展示框
  - 墨水瓶：青色染色玻璃板

#### **边界标识** (广场四周)

- **破损石柱**：每10格一个
  - 高度：3-4格随机
  - 材质：裂石砖+苔石砖柱
  - 顶部：可能放置火把或空无
- **简易长椅**：
  - 结构：橡木台阶（座位）+ 橡木栅栏（腿）
  - 数量：4个，面向篝火
  - 状态：部分破损（移除一段栅栏）

### 2. 破损的棚屋 (共6个，分散广场周围)

#### **标准棚屋设计** (5×7×3)

```
平面图：
[墙][门][墙]
[     内      ]
[墙]    [墙]
```

- **地基**：

  - 材质：裂石砖+圆石混合
  - 形状：不完全规则，部分角落凹陷
  - 高度：比地面高半格（台阶）
- **墙体**：

  - 框架：烧焦的橡木原木（四角）
  - 墙面：云杉木板（随机替换为破损橡木木板）
  - 破损处：使用楼梯、半砖制造缺口
  - 窗户：1×1，无玻璃，偶有铁栏杆
- **屋顶**：

  - 材质：云杉木楼梯（主）+ 石砖楼梯（边缘加固）
  - 坡度：前后两向倾斜
  - 破洞：随机移除1-2个方块，露出内部
- **内部装饰**（每棚屋选择1-2种）：

  - **类型A - 存储棚**：

    - 箱子x2（一个正常，一个破损）
    - 物品展示：腐肉、木棍、燧石
    - 地面：干草捆、麻袋（棕色地毯）
  - **类型B - 工具棚**：

    - 工作台（破损，使用铁砧重命名）
    - 熔炉（熄灭状态）
    - 墙上：工具架（物品展示框展示石镐、石斧）
  - **类型C - 居住棚**：

    - 床（红色床，破损-移除一个方块）
    - 床头柜：橡木压力板+花盆（枯萎灌木）
    - 小桌：橡木栅栏+橡木压力板
  - **类型D - 空置棚**：

    - 蜘蛛网若干
    - 灰尘：灰色地毯散落
    - 老鼠洞：1×1洞，深色橡木活板门遮盖

#### **特殊棚屋 - 老守卫的住处**

- **位置**：最靠近布雷恩位置
- **额外特征**：
  - 门外：生锈的铁盔甲架（无头盔）
  - 墙上：破损盾牌（物品展示框）
  - 内部：简易地图桌（橡木压力板+纸物品展示框）
  - 角落：武器架（铁剑、弓展示）

### 3. 残缺的石雕 (5-8个，散布河谷)

#### **类型一：断裂的方尖碑** (3个)

- **完整高度**：原应8-10格
- **现存部分**：随机3-6格高
- **材质**：
  - 主体：磨制安山岩+闪长岩混合
  - 基座：裂石砖
  - 断裂面：使用楼梯制造粗糙效果
- **符文痕迹**：
  - 侧面：紫珀块镶嵌（1-2个，表示残存能量）
  - 地面：围绕符文石（磨制安山岩压力板）

#### **类型二：人形雕像残骸** (2个)

- **底座**：2×2裂石砖
- **残存部分**：
  - 腿部：石砖楼梯堆叠
  - 躯干碎片：散落的石砖、石台阶
  - 头部：单独放置的石砖（可选）
- **设计要点**：
  - 避免完整人形，暗示被摧毁
  - 周围散落碎石（圆石、安山岩）

#### **类型三：神秘图腾柱** (2-3个)

- **结构**：
  - 主干：烧焦的深色橡木原木（3-4格高）
  - 雕刻：使用不同朝向的楼梯、半砖制造图案
  - 顶部：可能放置骷髅头（装饰头颅）
- **特点**：
  - 相对完整，暗示受保护
  - 底部有祭品痕迹：骨块、凋零玫瑰

#### **分布逻辑**：

- 2个在广场边缘（对称）
- 2-3个沿主路径分布
- 1个在森林入口附近
- 1个在方尖碑视线范围内（形成呼应）

### 4. 方尖碑及祭坛 (关键建筑)

#### **选址要求**：

- 距广场40-60格
- 地势略高（人工抬升3-5格）
- 周边10格内无高大树木遮挡

#### **祭坛平台** (15×15)

- **层级设计**：
  ```
  最外层：裂石砖台阶（1格宽）
  中间层：磨制安山岩（2格宽）
  内层：紫珀块（1格宽）
  中心：方尖碑基座
  ```
- **阶梯**：四方向各3格宽阶梯
- **符文阵**：
  - 地面镶嵌：紫珀块+末地烛（低亮度照明）
  - 图案：简单几何图形（方形套圆形）

#### **方尖碑本体** (7×7×25)

```
剖面结构：
顶部：能量核心 (1层)
碑体：主体部分 (20层)
基座：符文阵列 (4层)
地下：隐藏石室 (4层)
```

- **基座层** (Y=0到3)：

  - 材质：哭泣的黑曜石（主）+ 黑曜石（边角）
  - 符文显示：紫珀块镶嵌成简单纹路
  - 隐藏石匣：一面墙上有1×1凹陷（石砖活板门）
- **碑体层** (Y=4到23)：

  - 主体：紫珀柱（垂直堆叠）
  - 表面纹理：每4格镶嵌一条紫珀块横带
  - 能量脉络：随机位置的紫珀块替换为荧石（发光）
  - 破损处：随机移除1-2个方块，用紫珀楼梯填补边缘
- **顶部** (Y=24)：

  - 能量核心：信标（紫色光束）+ 紫水晶块（装饰）
  - 保护框架：紫珀块围成3×3框架
  - 粒子发射点：核心上方2格处放置紫水晶母岩（隐藏）
- **地下石室** (Y=-1到-4)：

  - 入口：基座侧面隐藏门（石砖活板门）
  - 通道：向下阶梯，紫珀块墙面
  - 密室：5×5×3空间
  - 内容：古老书籍展示（讲台+书）、能量收集装置（酿造台+玻璃瓶）

#### **脉冲效果实现**：

1. **视觉脉冲**：

   - 紫珀块亮度周期性变化（0-15）
   - 粒子效果：紫色彩色粒子从基座向上螺旋
   - 光束强度：信标光束亮度脉动
2. **音效区域**：

   - 以方尖碑为中心半径20格区域
   - 播放心跳音效，音量随距离衰减
3. **交互点**：

   - 基座四周4个压力板（磨制安山岩）
   - 右键触发动画序列

## 三、建筑技术实现细节

### 1. 材质替换表


| 建筑部位 | 主材质 | 破损材质      | 装饰材质    |
| -------- | ------ | ------------- | ----------- |
| 墙体     | 石砖   | 裂石砖(30%)   | 苔石砖(10%) |
| 木材     | 橡木   | 烧焦橡木(20%) | 云杉木(30%) |
| 地面     | 草径   | 灰化土(40%)   | 砂土(20%)   |
| 能量     | 紫珀   | 紫水晶(10%)   | 荧石(5%)    |

### 2. 破损效果实现技巧

- **随机移除**：每面墙随机移除2-5个方块
- **楼梯应用**：使用楼梯方块制造边缘破损
- **半砖组合**：上下半砖混合产生参差感
- **植物侵蚀**：藤蔓、苔藓覆盖下部
- **蜘蛛网**：角落放置1-2个蜘蛛网

### 3. 照明系统分级

- **安全区**（光照≥12）：火把、灯笼规则布置
- **过渡区**（光照8-11）：火把间距拉大
- **危险区**（光照≤7）：仅天然光照，偶有荧石
- **特殊照明**：符文火把（实际为海晶灯+染色玻璃）

## 四、建筑制作流程

### 第一阶段：地形准备 (1-2天)

1. 确定河谷边界（直径200-300格）
2. 地形塑造：
   - 中央平坦区（广场位置）
   - 缓坡过渡至森林
   - 人工河道痕迹（干涸河床）
3. 基础植被：
   - 稀疏橡树（烧焦效果）
   - 枯草丛（枯萎灌木）
   - 地面碎石（安山岩散落）

### 第二阶段：核心建筑 (3-4天)

1. 中央广场 + 篝火
2. 方尖碑 + 祭坛
3. 6个棚屋（不同功能）
4. 石雕布置

### 第三阶段：功能区建筑 (2-3天)

1. 森林入口 + 采集者营地
2. 尘封小径路障
3. 职业建筑框架（先完成外部）

### 第四阶段：细节装饰 (2-3天)

1. 破损效果添加
2. 物品摆放（环境叙事）
3. 照明系统布置
4. 粒子效果设置点

### 第五阶段：测试调整 (1天)

1. 玩家动线测试
2. 光照检查（夜间测试）
3. 怪物生成点验证
4. 事件触发区域确认

## 五、性能优化建议

### 1. 实体数量控制

- 物品展示框限制：每个建筑≤10个
- 盔甲架限制：仅在关键位置使用
- 活体装饰：避免使用过多生物

### 2. 渲染优化

- 使用屏障方块隐藏远处建筑细节
- 粒子效果距离衰减：50格外停止渲染
- 复杂结构内部使用简单材质

### 3. 区块加载策略

- 广场区域保持常加载
- 外围建筑按需加载（玩家接近时）
- 事件区域预先加载（事件开始前5分钟）

### 4. 备用简化方案

- 高延迟时：减少粒子密度50%
- 低配客户端：关闭薄雾效果
- 多人场景：简化Boss战区域装饰
